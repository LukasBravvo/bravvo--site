# THE BRAVVO — Site estático

Versão pequena do site com HTML/CSS/JS separados. Feito para publicar rapidamente no GitHub Pages, Vercel ou Netlify.

Como usar localmente
1. Coloque os arquivos em uma pasta:
   - index.html
   - css/styles.css
   - js/app.js
2. Abra `index.html` no navegador (ou rode um servidor local como `live-server` ou `python -m http.server`).

Publicar no GitHub Pages (rápido)
1. Crie um repositório no GitHub e envie os arquivos (`git push`).
2. Nas configurações do repositório (Settings > Pages), escolha a branch `main` (ou `gh-pages`) e a pasta `/ (root)`.
3. Aguarde alguns minutos e acesse `https://<seu-usuario>.github.io/<seu-repo>/`.

Publicar no Vercel
1. Crie conta em https://vercel.com e conecte seu repositório.
2. O Vercel detecta sites estáticos automaticamente e faz deploy.
3. Pode customizar domínio ou usar o domínio gratuito do Vercel.
